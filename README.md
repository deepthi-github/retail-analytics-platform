# Retail Analytics Platform

## Overview

This repository contains a Master's group project completed as part of the MSc Data Science programme at Newcastle University.

The project explores how data can be used to improve retail decision-making for an ice cream company operating in France. The proposed solution focuses on integrating multiple data sources to support demand forecasting, inventory management, freezer placement, and business performance monitoring through role-based dashboards.

The repository includes:
- Executive project report
- Dashboard prototype
- Business and technical solution design

---

## Business Problem

The company faced several operational challenges during peak sales seasons, including:

- Difficulty forecasting product demand
- Inefficient inventory management
- Poor freezer placement decisions
- Limited visibility into regional sales performance
- Lack of data-driven decision making

The objective was to design an analytics solution that enables stakeholders to make informed operational and strategic decisions.

---

## Proposed Solution

The proposed solution is a role-based retail analytics dashboard that combines business data with external factors such as:

- Sales data
- Weather information
- Tourism trends
- Footfall data
- Local events
- IoT freezer monitoring data (concept)

The dashboard provides insights for different business users including executives, regional managers, logistics managers, and shop owners.

---

## Dashboard Features

The dashboard prototype includes:

- Executive KPI dashboard
- Regional performance monitoring
- Revenue and demand forecasting
- Inventory monitoring
- Freezer performance tracking
- Weather and footfall analysis
- Event-based demand insights
- AI-assisted business reporting (concept)
